## Annotation Standards

1. **Spacing**: Label all margins and padding
2. **Typography**: Include font size, weight, color
3. **Colors**: Use hex codes, not names
4. **States**: Show normal, hover, active, disabled
5. **Interactions**: Arrow indicators for navigation
6. **Notes**: Specs and notes in comment bubbles

## Export Standards

- Format: PNG or SVG
- Quality: High resolution (2x for retina)
- Background: White or transparent
- Include all states and variations

## User Research Insights

### Primary Users
- Teachers: Need quick grading interface
- Students: Need clear assessment progress
- Parents: Need attendance and grade visibility
- Admin: Need comprehensive reporting

### Key Usability Findings
- Minimize steps to access assessments (max 3 clicks)
- Show real-time notifications
- Mobile-first design critical
- Dark mode consideration for evening studying
