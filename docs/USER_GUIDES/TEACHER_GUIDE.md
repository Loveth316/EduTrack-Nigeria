# Teacher Guide - EduTrack Nigeria

## Overview

Teachers use EduTrack Nigeria to manage their classes, create assessments, input student scores, track attendance, and generate reports.

## Getting Started

### Login
1. Go to EduTrack Nigeria website
2. Click "Login"
3. Enter email and password
4. Click "Sign In"

### Dashboard
Your dashboard shows:
- Classes you teach
- Pending assessments
- Recent results
- Attendance summary
- Quick actions

## Class Management

### View Class
1. Click **Classes** from menu
2. Select your class
3. View:
   - Class roster
   - Class schedule
   - Students

### View Class Roster
1. Go to **Classes > [Class Name]**
2. Click **Students**
3. View all enrolled students
4. Search for specific student

## Assessment Management

### Create Assessment
1. Go to **Assessments > Create New**
2. Fill in details:
   - Title (e.g., "Mathematics Quiz")
   - Type: CA1, CA2, CA3, or Exam
   - Subject
   - Class
   - Total Score
   - Due Date
3. Click **Create**

### Upload Assessment Materials
1. Go to **Assessments > [Assessment Name]**
2. Click **Upload Materials**
3. Attach:
   - Questions/Exam paper
   - Instructions
   - Solution key (optional)
4. Save

### View Submissions
1. Go to **Assessments > [Assessment Name]**
2. Click **Submissions**
3. View:
   - Student name
   - Submission status
   - Submission time
   - Files submitted

## Grading

### Input Individual Score
1. Go to **Results > Input Scores**
2. Select:
   - Subject
   - Assessment
   - Class
3. Click on student name
4. Enter score
5. Add feedback (optional)
6. Click **Save**

### Bulk Input Scores
1. Go to **Results > Bulk Input**
2. Select assessment
3. Download spreadsheet
4. Fill in scores
5. Upload spreadsheet
6. Preview and confirm

### Provide Feedback
1. Go to **Results > [Assessment]**
2. Click on student score
3. Add feedback comments
4. Save feedback
5. Student receives notification

## Attendance

### Mark Attendance
1. Go to **Attendance > Mark Attendance**
2. Select:
   - Date
   - Class
3. Check status for each student:
   - ✓ Present
   - ✗ Absent
   - ≈ Late
4. Add remarks if needed
5. Click **Submit**

### View Attendance Report
1. Go to **Attendance > Report**
2. Select date range
3. Choose class
4. View attendance percentage per student
5. Export as PDF

### Send Attendance Notification
1. Go to **Attendance > Notifications**
2. Select students with low attendance
3. Choose notification type
4. Send message to students/parents

## Lesson Notes

### Create Lesson Note
1. Go to **Lesson Notes > New**
2. Enter:
   - Title
   - Subject
   - Class
   - Content
3. Upload media (images, videos, audio)
4. Click **Publish**

### Upload Media
1. Click **Add Media**
2. Upload:
   - Images
   - Videos
   - Audio files
   - PDF documents
3. Add descriptions
4. Save

## Report Cards

### Generate Report Card
1. Go to **Reports > Report Cards**
2. Select:
   - Term
   - Class
   - Subject (if filtering)
3. Click **Generate**
4. Review scores and grades

### Add Teacher Comments
1. Go to **Reports > Report Cards**
2. Click on report card
3. Click **Edit Comments**
4. Add your assessment of student's performance
5. Save comments

### Export Report Card
1. Go to **Reports > Report Cards**
2. Select report card
3. Click **Export**
4. Choose format:
   - PDF
   - Excel
5. Download file

## Communication

### Send Message to Student
1. Go to **Messages > New**
2. Select student(s)
3. Type message
4. Click **Send**

### Send Message to Parent
1. Go to **Messages > New**
2. Select parent
3. Type message
4. Click **Send**

## Tips & Best Practices

1. **Regular Updates**: Input grades regularly, not all at once
2. **Detailed Feedback**: Provide constructive comments to help students
3. **Clear Instructions**: Always include clear assignment instructions
4. **Timely Grading**: Grade assessments within 1 week
5. **Documentation**: Keep records of all assessments and scores

## Troubleshooting

**Issue**: Cannot submit grades
- Solution: Ensure assessment is selected and scores are within valid range

**Issue**: Students cannot access assignments
- Solution: Check assessment publication date

**Issue**: Attendance not saving
- Solution: Ensure all required fields are filled before submitting
